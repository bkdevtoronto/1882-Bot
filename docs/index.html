1882Bot is a reddit moderator bot with a suite of customisable tools for use on reddit.

Download Now

Download now!

Run the Python app in your desktop or server.

1. Download

Download from GitHub, or from your command line:
$ git clone https://github.com/bkdevtoronto/1882-Bot

Download PRAW (Python reddit API Wrapper):
$ pip install praw

2. Configure PRAW

Open up praw.ini to add in the following:

  • Your bot's username and password
  • Your app's secret and key

If you don't have an app, you can create one by visiting reddit/prefs/apps. Ensure it is created as a script type in your bot's account.

3. Configure 1882-Bot

Finally, open up the configuration file at config.py where you can add your bot's details, sub details, and turn on and off features.

You can even customise the messages, flairs and other actions that 1882-Bot takes.
